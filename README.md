 ## Trabalho em dupla - Miguel Francelino Santos e Melissa Leite Borba

**O que é o projeto:**
Neste projeto, foi desenvolvido uma plataforma simples porém funcional de uma concessionária de veículos, onde a interface deve ser de fácil uso para quando necessário, o estoque ser consultado e que forneça as informações essenciais para venda dos automóveis.

**Como foi feito:**
No nosso sistema web, começamos pelo HTML e CSS, montando a parte dos carros onde podem visualizar cada um com seu respectivo nome, preço e imagem. Adicionamos o JavaScript para ficar funcional e intuitivo para o vendedor ou comprador. Após isso,  repetimos a com as motos e fizemos o mural dos vendedores com suas fotos e nomes para os compradores saberem a quem recorrer e personalizamos da forma que achamos visualmente mais bonito.

**Funcionalidades:**
Suas funcionalidades implementadas, são as API's para a geração de cards com imagens, nomes e preços de cada automóvel ou vendedor.
